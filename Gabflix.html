Projeto Gabflix

Clone do Netflix com Tema Studio Ghibli

Um projeto de streaming inspirado na Netflix, focado nos filmes do Studio Ghibli

Visão Geral

O Gabflix é um projeto que recria a interface do Netflix, adaptando-a para apresentar o catálogo do Studio Ghibli. Desenvolvido com HTML, CSS e JavaScript, o projeto demonstra habilidades em:

  • Layout responsivo com CSS Grid e Flexbox
  • Animações suaves e transições
  • Carrossel de imagens com Swiper.js
  • Sistema de navegação dinâmico
  • Interface de usuário moderna e intuitiva

Tecnologias Utilizadas

HTML5
CSS3
JavaScript
Responsive
